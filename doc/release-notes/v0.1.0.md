# Extreme Anglin' Atlas v0.1.0

Release date: 26MAY2026

Extreme Anglin' Atlas is an in-game fishing reference for World of Warcraft TBC Classic. This first release focuses on making fishing locations easy to browse by zone, region, and fish.

## Highlights

- Browse fishing catches by Eastern Kingdoms, Kalimdor, Outland, and other tracked regions.
- Select a zone to see available fish and catch skill ranges.
- Search by fish name to find every known zone where that fish can be caught.
- View item icons and in-game item tooltips where item data is available.
- Open the addon with `/extremeanglinatlas` or `/eaa`.

## Release Details

- Packaged as a CurseForge-ready zip with a top-level `ExtremeAnglinAtlas/` addon folder.
- Includes MIT license text in the release package.
- Includes addon thumbnail/icon metadata.
- Categorized as `Professions` in the addon TOC.

## Known Limitations

- Fishing data is based on the included source workflow and manual corrections, so some locations may still need refinement over time.
- Item names, quality colors, icons, and tooltips depend on what the game client has available through its item cache.
