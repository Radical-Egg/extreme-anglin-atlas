# Changelog

All notable changes to Extreme Anglin' Atlas will be documented in this file.

This project follows semantic versioning where practical.

## [0.1.0] - TBD

### Added

- Initial TBC Classic addon release.
- Browse fishing data by region and zone.
- Search for fish and view every known zone where each fish can be caught.
- Display fish item icons, item tooltips, and catch skill ranges where available.
- Slash commands: `/extremeanglinatlas` and `/eaa`.
- Generated addon runtime data from normalized fishing location sources.
- CurseForge-ready packaging with a top-level `ExtremeAnglinAtlas/` addon folder.
- Addon icon, MIT license metadata, and Professions category metadata.

### Notes

- The packaged release includes `LICENSE` and `Thumbnail.PNG`.
- Runtime data is generated from the CSV data workflow; update source data and rebuild rather than editing generated files directly.
